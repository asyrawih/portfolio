# Program Pertama Golang

<Steps>
### Instalasi Golang

untuk instalasi golang silahkan running di terminal kesayangan anda
di sini saya mengunakan iterm 

```bash showLineNumbers
$ curl -s https://raw.githubusercontent.com/golang/dep/master/install.sh | sh
```

atau anda bisa mengunjungi download file [di sini](https://golang.org/dl/) untuk menginstall golang
untuk melihat versi golang yang di install silahkan pilih menu [go](https://golang.org/dl/)

### Pilih Editor Ke Sukaan Anda 
umum nya vscode tapi anda bisa mengunankan editor lain

### Selesai
<Callout emoji="ℹ️">
